# Weekly Report Skill

> 将工作流水账自动整理成标准周报格式，输出为 Markdown 表格，包含「上周工作总结」和「本周工作计划」两个部分。

## 技能简介

本技能帮助你把一周的工作记录（流水账）快速整理成规范的周报格式。只需粘贴你的工作记录，AI 会自动提取关键信息，按三只青蛙（OKR 优先级）排序，生成结构化的 Markdown 周报。

### 核心特性

- **两表结构**：上周工作总结 + 本周工作计划，完整覆盖周报需求
- **三只青蛙排序**：按 OKR 优先级自动排序，🐸1 > 🐸2 > 🐸3
- **状态颜色标记**：🟢 完成 / 🟡 部分完成 / 🔴 未完成
- **难度系数评估**：⭐~⭐⭐⭐，2星和3星自动提示填写核心措施
- **智能补全**：信息不足时自动标注「待补充」并列出缺失字段
- **年度复盘**：提供年度复盘模板，支持全年 OKR 完成情况总结、经验教训、下一年展望

## 安装方法

```bash
npx skills add https://github.com/wrack0001/milian --skill weekly-report
```

## 使用方式

直接粘贴你的工作记录，说明需要生成周报即可：

```
帮我把这周的工作整理成周报：
- 完成了用户认证模块开发，包括 JWT 登录、权限校验
- 修复了 3 个线上 bug
- 参加了产品需求评审会议
- 下周计划做数据导出功能
```

或者更简单：

```
帮我写周报（粘贴流水账）
```

## 周报格式预览

### 上周工作总结

| 三只青蛙排序 | KR及其进展 | 上周重要任务 | 原期望的结果是什么 | 实际产出/进展 | 状态说明 | 补充说明 |
|:---:|:---|:---|:---|:---|:---:|:---|
| 🐸1 | 用户系统 | JWT 登录<br>权限校验 | 完成认证模块 | 完成 JWT 登录和权限校验<br>【3.18～3.24】（3pd）（已上线） | 🟢 | |

### 本周工作计划

| 三只青蛙排序 | KR | 本周重要任务 | 期望的结果 | ⭐难度系数 | 核心措施 | 补充说明 |
|:---:|:---|:---|:---|:---:|:---|:---|
| 🐸1 | 数据导出 | 数据导出功能开发 | 支持 Excel/CSV 导出 | ⭐⭐ | 调研现有导出库，复用已有数据查询接口 | |

## 字段说明

### 三只青蛙

"三只青蛙"来自时间管理方法论——每天最重要的三件事。在周报中：
- 🐸1：最重要的 KR/项目
- 🐸2：次重要的 KR/项目  
- 🐸3：第三重要的 KR/项目
- 其余：临时任务，不加青蛙图标

### 状态说明

| 颜色 | 含义 | 是否进入下周计划 |
|:---:|:---|:---:|
| 🟢 | 按计划完成 | 否 |
| 🟡 | 部分完成或有风险 | 是 |
| 🔴 | 未完成或严重延期 | 必须 |

### 难度系数

| 星级 | 含义 | 是否需要核心措施 |
|:---:|:---|:---:|
| ⭐ | 简单，常规任务 | 否 |
| ⭐⭐ | 中等，有一定挑战 | 是 |
| ⭐⭐⭐ | 困难，需要重点关注 | 是 |
