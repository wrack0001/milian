## 字段说明

| 字段 | 说明 | 取值 |
|:---|:---|:---|
| 优先级 | 三只青蛙排序 | 🐸1 / 🐸2 / 🐸3 |
| KR 标题 | 关键结果名称，必须唯一 | 简短文字 |
| 目标 | 本 OKR 要达成的目标 | 一句话描述 |
| 关键结果 | 可量化的结果，多条用 `<br>` 分隔 | 列表 |
| 负责人 | 负责人姓名/团队 | 文字 |
| 截止日期 | 格式 MM-DD | 如 03-31 |
| 状态 | 当前执行状态 | ⚪计划中 / 🟡进行中 / 🟢已完成 / ⏸️暂停 |
| 进度 | 完成百分比 | 如 60% |
| 备注 | 当前进展说明，可留空 | 文字 |

## 使用说明

1. **季度初**：运行 `scripts/new-quarter.sh` 自动创建此文件，然后修改内容填入本季度 OKR
2. **季度中**：每周更新「进度」和「备注」列，状态变化时更新「状态」列
3. **KR 标题必须唯一**：Skill 用标题匹配周报中的任务
4. **提交 Git**：每次修改后提交，Git 历史自动记录演变
